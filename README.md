# ultralytics-disk-cache-hook

Redirect `ultralytics` `cache="disk"` image caches away from the dataset directory and into a local temporary cache root.

将 `ultralytics` 的 `cache="disk"` 图片缓存从数据集目录重定向到本地临时缓存目录。
